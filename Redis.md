# Redis

**Redis**(REmote DIctionary Server) - NoSQL база данных In-memory. Используется как бд или кэш.
---
Особенности:  
* **Скорость:** За счет работы в раме она очень быстрая. Способна обрабатывать до 100000 запросов в секунду.
* **5 типов данных:** В отличие от memcached поддерживает не только строки а Strings, Lists, Sets, Hashes, Zsets.
*  **Атомарность:** Все операции выполняются в один поток, что исключает гонку за один обьект.
---
# Команды: 

**Keys: **

**Set key value** - создать/обновить ключ  
**Get key** - прочитать значение  
**Del key** -  удалить ключ  
**Exists key** - проверить существует ли ключ   

---
1 - успешно(true)
0 - false
