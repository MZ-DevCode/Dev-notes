# 🐙 GitHub & Git Cheat Sheet

> **Git** — система контроля версий (локально).  
> **GitHub** — облачный сервис для хранения ваших Git-репозиториев.

---

## 🚀 Основной рабочий цикл (Workflow)

Когда вы изменили файл и хотите отправить его в облако:

1.  **Проверить статус:** `git status` (покажет, какие файлы изменились).
2.  **Подготовить:** `git add .` (добавить все изменения) или `git add filename.md`.
3.  **Сохранить (Коммит):** `git commit -m "описание того, что сделал"`.
4.  **Отправить:** `git push origin main`.

---

## 🛠 Полезные команды


| Команда | Описание |
| :--- | :--- |
| `git init` | Инициализировать новый репозиторий в папке. |
| `git clone <url>` | Скачать чужой проект к себе на компьютер. |
| `git pull` | Забрать свежие изменения из облака (если кто-то другой их туда залил). |
| `git log` | Посмотреть историю всех сохранений (коммитов). |
| `git diff` | Посмотреть, что именно изменилось в коде с прошлого раза. |
