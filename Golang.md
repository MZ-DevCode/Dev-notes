# 🔵 Go (Golang)

> **Go** — это компилируемый язык от Google с открытым исходным кодом. Он сочетает производительность C++ с простотой Python и встроенной поддержкой многопоточности.

---

## ⚡️ Ключевые особенности

*   **Статическая типизация:** Ошибки типов ловятся при компиляции.
*   **Компиляция в бинарник:** Программе не нужен интерпретатор; она собирается в один исполняемый файл.
*   **Concurrency (Горутины):** Легковесные потоки (`goroutines`) позволяют запускать миллионы параллельных задач.
*   **Строгий стиль:** Неиспользуемые переменные или импорты приводят к ошибке компиляции.

## 🛠 Базовый синтаксис (Cheat Sheet)

### 📦 Переменные и типы данных (Подробно)

В Go строгая статическая типизация. Переменную нельзя переопределить другим типом данных после объявления.

#### 1. Объявление переменных

| Способ | Синтаксис | Особенности |
| :--- | :--- | :--- |
| **Полный** | `var name string = "Go"` | Можно объявлять на уровне пакета (вне функций). |
| **Краткий** | `name := "Go"` | **Только внутри функций**. Тип определяется автоматически. |
| **Групповой**| `var ( a = 1; b = 2 )` | Удобно для объявления нескольких настроек сразу. |

#### 2. Нулевые значения (Zero Values)
Если вы объявили переменную, но не дали ей значение, Go присвоит ей «дефолт»:
*   `int`, `float` → `0`
*   `string` → `""` (пустая строка)
*   `bool` → `false`
*   `указатели, интерфейсы, слайсы` → `nil`

#### 3. Константы (Constants)
Константы вычисляются на этапе компиляции. Им нельзя присвоить результат работы функции (например, время).
```go
const Pi = 3.14
const (
    StatusOk = 200
    StatusError = 500
)

// Использование iota (авто-инкремент для перечислений)
const (
    A = iota // 0
    B        // 1
    C        // 2
)
```
#### 4. Целочисленные типы (Integers)


| Тип | Размер | Описание |
| :--- | :--- | :--- |
| `int` | 32 или 64 бит | Зависит от платформы (рекомендуется по умолчанию) |
| `uint` | 32 или 64 бит | Беззнаковое целое (только положительные) |
| `int8` .. `int64` | 8 - 64 бит | Числа со знаком с фиксированным размером |
| `uint8` .. `uint64`| 8 - 64 бит | Числа без знака с фиксированным размером |
| `byte` | 8 бит | Псевдоним (alias) для `uint8` |
| `rune` | 32 бит | Псевдоним для `int32` (хранит один символ Unicode) |

#### 5. Числа с плавающей точкой (Floating Point)


| Тип | Точность | Описание |
| :--- | :--- | :--- |
| `float32` | 7 знаков | Стандарт IEEE-754, 32 бита |
| `float64` | 15 знаков | Рекомендуется использовать по умолчанию (64 бита) |
| `complex64` | - | Комплексное число (float32 для реальной и мнимой частей) |
| `complex128`| - | Комплексное число (float64 для частей) |

#### 6. Логический тип (Boolean)


| Тип | Значения | Особенности |
| :--- | :--- | :--- |
| `bool` | `true`, `false` | Нельзя привести к `int` (0 или 1) напрямую |

#### 7. Строки (Strings)


| Тип | Описание | Особенности |
| :--- | :--- | :--- |
| `string` | Набор байт | **Immutable** (неизменяемые). Хранят текст в UTF-8. |

   // c := a + b // ОШИБКА
